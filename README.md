# MediGuide AI 🩺

## Agentic AI Health & Wellness Assistant

MediGuide AI is an AI-based solution aligned with SDG 3: Good Health and Well-being.

## Problem
Many people need quick health guidance and wellness support.

## Solution
MediGuide AI uses multiple AI agents:

- Coordinator Agent
- Symptom Agent
- Mental Wellness Agent
- Lifestyle Agent
- Health Education Agent

## Features

- User health queries
- Agent selection
- Wellness guidance
- Simple health education

## Technology

- Python
- Streamlit
- Generative AI concepts

## Impact

Supports better health awareness and promotes preventive healthcare.
